# Prismic Eloquent
Use the Prismic Api in a more (friendly) eloquent like way.

- Runs on the latest Prismic api (v2)
- Easy query writing
- Relationship resolving with hasOne and hasMany
- Support eager loading with the Prismic fetchLinks
